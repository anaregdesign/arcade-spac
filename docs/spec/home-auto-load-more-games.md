# Home Auto Load More Games

## Summary

Home のゲーム一覧末尾に近づいたとき、次のゲームカード群を自動で読み込み、手動の `Show more games` 操作なしで catalog を続けて見られるようにする。

## User Problem

- 一覧の続きを見るたびに手動 button を押す必要があると、catalog を流し見しづらい
- ゲーム数が増えるほど、探索中の中断が増える
- 検索や絞り込み後も、同じ文脈のまま続きを見たい

## Users and Scenarios

- プレイヤーは Home を下へスクロールしながら、自然に次のゲーム群へ進みたい
- プレイヤーは検索や絞り込みを変えたあとも、新しい一覧の先頭から同じように探索したい
- 自動読込が使えない場合でも、一覧の続きを自力で開きたい

## Scope

- 一覧末尾へ近づいたら追加のゲームカードを段階的に自動表示する
- 検索、絞り込み、並び替え変更後は一覧の先頭状態へ戻す
- 自動読込が使えないときの fallback を残す

## Non-Goals

- Home card デザインの変更
- サーバー側 pagination の導入
- 並び順ロジックの変更

## User-Visible Behavior

- Home を下方向へスクロールして末尾へ近づくと、次のゲームカード群が自動で見える
- 検索、絞り込み、並び替えを変えたあとは、先頭状態から再び探索を始められる
- 自動読込が働かない環境でも、手動で続きを開ける

## Acceptance Criteria

- 一覧末尾へ近づいたとき、ゲームカードが段階的に追加表示される
- 初期表示直後の意図しない自動読込で一覧が暴発しない
- 検索、絞り込み、並び替え変更後の一覧 reset が分かりやすい
- Fallback の手動読込でも、同じ続きを開ける

## Edge Cases

- 一覧が短くても、勝手に何度も追加読込されない
- 6 件以下の結果しかないときは、余計な追加導線を見せなくてよい
