# Spinner Aim

## Summary

回転する launcher の角度を見極めて projectile を target ring へ通す precision action game。

## User-Visible Behavior

- launcher が一定または変速で rotate する
- 利用者は correct angle で fire する
- target ring hit で success、hazard hit で penalty

## Acceptance Criteria

- 2 分以内の shot-based run
- charge は不要で aim timing が主題
- Result に targets hit と accuracy が出る

## Distinction

- Precision Drop の vertical timing ではなく angular shot timing を扱う
