# Home Recommendation Ranking

## Summary

Home のゲーム一覧は、最近のプレイ傾向や継続率を反映して、次に遊びたくなるゲームが上位に見えやすい順序へ寄せる。

## User Problem

- 固定順の一覧だけでは、次に遊びたいゲームが見つけにくい
- 完走、再挑戦、共有のような前向きな行動が次回の発見体験に反映されない
- 途中離脱が多いゲームまで同じ優先度で並ぶと、探索の質が上がりにくい

## Users and Scenarios

- プレイヤーは Home を開いたとき、自分に合いそうなゲームが上位に見えていてほしい
- 継続利用者は、最近のプレイ傾向が次回の並び順に反映されていてほしい
- 新規利用者でも、学習が足りない間は安定した一覧からゲームを探したい

## Scope

- Home のゲーム一覧順を、プレイ完走、再挑戦、共有などの行動を踏まえて変化させる
- 学習が十分でないときは安定した fallback 順を維持する
- 特定のゲームを隠すのではなく、見つけやすい順番を調整する

## Non-Goals

- Home card の layout 変更
- ゲームルールや score 計算の変更
- 外部広告や外部トラッキング data の利用

## User-Visible Behavior

- Home のゲーム一覧は、利用傾向に応じて上位に見えるゲームが変わる
- よく完走したり再挑戦したり共有したゲームに近い候補が、次回の Home で見つけやすくなる
- 学習が足りない時期でも、一覧が不安定に揺れず安定した fallback 順で見られる
- どのゲームも Home から辿れなくならず、順序だけが変わる

## Acceptance Criteria

- Home の並び順がプレイ傾向を受けて変化する
- 学習が浅い利用者や新規利用者でも、安定した fallback 順で探索できる
- 順位変化によって一部ゲームが見つけにくくなりすぎない

## Edge Cases

- 利用履歴がほとんどない時期は、固定に近い並びでも問題ない
- 一時的な行動の偏りで一覧が極端に入れ替わり続けない
